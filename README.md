# お手伝いポイント表

## 日常のお手伝い

| お手伝い内容 | 基本ポイント | 追加ボーナス条件 | ボーナスポイント |
|------------|------------|----------------|----------------|
| 洗い物 | 10 | 水筒を洗った場合 | +10 |
| 洗濯物を畳む | 40 | 自分以外の分も畳んだ場合 | ×1.5倍 |
| 掃除機がけ | 100 | - | - |
| クイックルワイパー | 50 | - | - |
| 除湿器の水捨て | 10 | - | - |
| 夜の水筒準備 | 30 | - | - |
| カーテン開け | 10/部屋 | 自分の部屋以外も開けた場合 | ×1.5倍 |

## ポイントの使い方

| 報酬内容 | 必要ポイント |
|---------|------------|
| スマホ使用 | 60P/1時間 |
| パソコン使用 | 四日間(R7/7/14～7/18)は使用は自由それ以降は(80P/1時間)です。 |

## ポイント計算方法

1. 基本ポイントを獲得
2. ボーナス条件を満たした場合は追加ポイントを加算
3. 獲得したポイント(P)は報酬と交換可能
